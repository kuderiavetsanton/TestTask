# Test Task

A simple web site with minimalistic design where you can create, edit, delete and comment posts

## Installation

Use the package manager [npm](https://www.npmjs.com/) to install dependencies and run Test Task.

```bash
npm i
npm run dev
```

## About

I used Next js as a SSR React framework,
Redux with a thunk as a data layer,
and Styled Components as a UI kit
